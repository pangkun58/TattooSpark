# TattooSpark 🔥

> AI 驱动的纹身设计平台

## 项目简介

TattooSpark 是一个 AI 驱动的纹身设计平台，帮助用户通过智能生成、个性化定制，找到属于自己的完美纹身方案。

## 核心功能

- 🎨 AI 纹身图案生成
- 🔍 风格探索与筛选
- 👤 用户账号系统（注册/登录）
- 💳 付费与充值功能
- 📱 纹身师匹配与预约

## 技术栈

- **前端**: React / Next.js
- **后端**: Node.js / Express
- **AI**: Stable Diffusion / DALL-E
- **数据库**: PostgreSQL
- **支付**: Stripe

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 项目结构

```
TattooSpark/
├── frontend/        # 前端应用
├── backend/         # 后端 API
├── ai/              # AI 生成模块
└── docs/            # 项目文档
```
